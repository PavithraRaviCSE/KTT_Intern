html - DOM
css - CSSOM

1. HTML Processing
Parser: HTML5 Parser (Converts HTML into a DOM Tree).

Engine: Rendering Engine (Renders the DOM visually).

2. CSS Processing
Parser: CSSOM Parser (Converts CSS into a CSSOM Tree).

Engine: Rendering Engine (Applies styles from CSSOM + DOM).

3. JavaScript Processing
Parser: JavaScript AST Parser (Converts JS into an Abstract Syntax Tree).

Engine: JavaScript Engine (V8, SpiderMonkey, JavaScriptCore) (Executes JavaScript code). 


Rendering Engine: Blick, WebKit, GecKo

iframe: 
    sandbox:
    Value	Effect
    allow-scripts	Enables JavaScript execution inside the iframe.
    allow-forms	    Allows form submissions inside the iframe.
    allow-popups	Allows the iframe to open new tabs or pop-ups.
    allow-same-origin	Allows iframe to access cookies and storage from its own domain.
    allow-modals	Allows modal dialogs inside the iframe.
    allow-downloads	Enables file downloads from the iframe.
