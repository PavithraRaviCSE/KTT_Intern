html - DOM
css - CSSOM

1. HTML Processing
Parser: HTML5 Parser (Converts HTML into a DOM Tree).

Engine: Rendering Engine (Renders the DOM visually).

2. CSS Processing
Parser: CSSOM Parser (Converts CSS into a CSSOM Tree).

Engine: Rendering Engine (Applies styles from CSSOM + DOM).

3. JavaScript Processing
Parser: JavaScript AST Parser (Converts JS into an Abstract Syntax Tree).

Engine: JavaScript Engine (V8, SpiderMonkey, JavaScriptCore) (Executes JavaScript code). 


Rendering Engine: Blick, WebKit, GecKo
